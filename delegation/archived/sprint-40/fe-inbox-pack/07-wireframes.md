# Wireframe (Textual)

```
┌──────────────────────────── Inbox ────────────────────────────┐
│ Filters: [type ▾] [tags ▾] [category ▾] [vault ▾] [sort ▾]    │
│ Actions: (Accept) (Archive) (Accept Selected) (Accept All)     │
├───────────────────────────────────────────────────────────────┤
│ [ ] 09:12  /type: link  tags: [#inbox]  title: "Interesting..."│
│     content: https://...                                       │
│     quick: [Accept] [Archive] [Edit] [Tag] [Type ▾] [Vault ▾]  │
├───────────────────────────────────────────────────────────────┤
│ [ ] 09:10  /type: todo  tags: [#inbox,#quickadd]               │
│     content: "Fix build script"                                │
│     quick: [Accept] [Archive] [Edit] [Tag] [Type ▾] [Vault ▾]  │
├───────────────────────────────────────────────────────────────┤
│ ...                                                            │
└───────────────────────────────────────────────────────────────┘
```

Keyboard:
- `↑/↓` move, `Enter` open, `A` accept, `X` archive, `T` tag, `Y` change type, `V` vault.
